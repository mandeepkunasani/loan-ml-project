Loan Prediction using Logistic Regression in Python with data preprocessing, feature encoding, and model evaluation.
# Loan Prediction ML Project

This project uses Machine Learning to predict loan approval status based on applicant details.

## 🔧 Technologies Used
- Python
- Pandas
- Scikit-learn

## 📊 Features
- Data cleaning and preprocessing
- Handling missing values
- Encoding categorical variables
- Logistic Regression model
- Model evaluation using accuracy and confusion matrix

## 📁 Files
- loan_project.py
- loan_data_set.csv

## 📌 Conclusion
The model predicts loan approval with good accuracy using Logistic Regression, making it suitable for binary classification problems.
